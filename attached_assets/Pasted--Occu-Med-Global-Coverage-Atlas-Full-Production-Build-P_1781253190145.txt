# Technical Requirements

## Stack

Use:

* React
* TypeScript
* Node.js
* Express
* PostgreSQL
* Neon database
* Prisma ORM or Drizzle ORM
* Tailwind CSS
* Framer Motion for animations

## Deployment

The application will be deployed on:

* Render

DO NOT:

* Use Replit-specific infrastructure
* Use Replit authentication
* Use Replit databases
* Use Replit-only APIs
* Hardcode localhost assumptions
* Create anything that only works inside Replit

The app must be fully portable and production deployable.

The project must:

* Run locally with npm install and npm run dev
* Build successfully with npm run build
* Use environment variables
* Include a clean .env.example
* Include Render deployment instructions
* Use DATABASE_URL for Neon
